A useful sequence

Start with evidence, not a package

A proposal becomes easier to evaluate when the same property facts and written specifications appear in every comparison.

  1. Document the symptom.Use safe ground-level photos and note when the issue appears.
  2. Map what feeds each run.Roof area, pitch, valleys, and run length shape the question.
  3. Trace outlets and downspouts.Identify transition points, restrictions, and each endpoint.
  4. Specify the proposed system.Record profile, size, material, hangers, fasteners, guards, and warranties.
  5. Compare equivalent scopes.Align inclusions and exclusions before comparing totals.
