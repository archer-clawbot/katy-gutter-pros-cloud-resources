A repeatable method

Start with evidence, not a package

Proposal comparison is more useful when each provider receives the same observations and every written scope describes the same work. The objective is not to design a system from a checklist. It is to expose gaps that require measurement, inspection, or written clarification.

  1. Document the symptom.Use safe ground-level photos and note where and when overflow, leakage, staining, sagging, or ponding appears. Avoid climbing during wet or windy conditions.
  2. Map what feeds each run.Roof area, pitch, valleys, inside corners, upper-roof discharge, and run length can concentrate flow. Ask the inspecting provider to show the measurements behind a recommendation.
  3. Trace outlets and downspouts.Record count, placement, dimensions, elbows, offsets, transitions, restrictions, and each visible endpoint. A gutter cannot be evaluated separately from the openings that drain it.
  4. Specify the proposed system.Capture profile, nominal size, material, finish, hangers, fasteners, attachment spacing, corners, guards, removal, disposal, cleanup, and responsibility for related repairs.
  5. Compare equivalent scopes.Align inclusions, exclusions, payment terms, change-order procedures, and written warranties before comparing totals. Ask for revisions when a material decision is only verbal.
